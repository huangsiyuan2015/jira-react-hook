## 项目描述

该项目是一个 PC 端应用——任务管理系统，功能模块划分为：

- 用户模块：用户注册、登录
- 项目模块：
  - 项目显示
    - 项目能够按照筛选条件过滤显示
  - 项目管理
    - 对项目进行添加/删除/修改/收藏
- 看板模块：
  - 看板显示：
    - 看板中的任务能够按照筛选条件过滤显示
  - 看板管理：
    - 对看板进行添加/删除，也可以拖动看板顺序
    - 在看板中添加/删除/修改任务，也可以拖动任务顺序
